🏧** ATM Simulator System (Java Desktop Application)**
📌 Project Overview

ATM Simulator System is a Java-based desktop banking application designed to simulate core ATM functionalities. The project is being developed using Java Swing for GUI and will integrate MySQL using JDBC for database operations.

Currently, the project includes the initial Login UI screen implementation.

🚀 Current Features (Phase 1)

Java Swing-based Login window

Custom GUI layout

Image integration (Bank logo, card image, background)

Structured project using Java packages

🛠️ Technologies Used

Java (Core Java)

Java Swing (GUI Development)

AWT (UI Components & Image Handling)

Intellij Idea

Command Line (Compilation & Execution)
